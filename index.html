Each photo — key rules
  • Same size & type in every shot — don't mix M6 bolts with M10 bolts in one image
  • Arrange items so they don't overlap (slight gaps are fine)
  • Set the item count stepper before you shoot — it embeds the count in the filename for labeling
  • Fill the frame — get the phone close
Across your session — vary these
  • Background: white paper, grey bench, wood, concrete — rotate through them
  • Lighting: bright overhead, dimmer indirect, near a window, shaded — mix it up
  • Angle: straight top-down, slight tilt (15–30°), on their side
  • Quantity: shoot groups of 1, 3, 5, 10, 15… per frame
  • Sizes: if you have multiple sizes of one class, include all of them
Targets
  • 200+ images per class minimum; 300+ for washers
  • Images upload to the training server automatically
  • Filename: bolt_0042_5x.jpg = bolt image #42, 5 items
  • If server is unreachable the image saves locally as a fallback
